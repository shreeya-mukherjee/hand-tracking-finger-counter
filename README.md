# Hand Tracking Finger Counter

A real-time hand tracking and finger counting system built using OpenCV and MediaPipe.

## Features

* Detects hands using MediaPipe.
* Tracks 21 hand landmarks.
* Counts raised fingers.
* Displays the total number of fingers in real time.
* Shows FPS (Frames Per Second).
* Supports detection of both hands.

## Technologies Used

* Python
* OpenCV
* MediaPipe

## Installation

```bash
pip install opencv-python mediapipe numpy
```

## Run the Project

```bash
python FingerCounter.py
```
